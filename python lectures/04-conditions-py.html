Conditions

Objectives

1. To know what are alternative paths to run a program

2. Performance improvement

  • Duration: 15 min
  • Exercise : 1
Sometimes in our program we have a situation that can be true or false. For example we are checking for positive and negative numbers. we want to deal with them with different conversion methods. Any given number may be positive or negative. To check it, we can use conditions. A condition can be evaluated either true or false and based on that outcome we can define flow of our program.

To use condition we need to know about comparison operators. They are used to compare two values.


Operator Description
==If the values of two operands are equal, then the condition becomes true.
!=If values of two operands are not equal, then condition becomes true.
>If the value of left operand is greater than the value of right operand, then condition becomes true.
<If the value of left operand is less than the value of right operand, then condition becomes true.
>=If the value of left operand is greater than or equal to the value of right operand, then condition becomes true.
<=If the value of left operand is less than or equal to the value of right operand, then condition becomes true.


simple if condition

suppose we have a variable and we want to check the type of variable.

x = 10 + 2j

if type(x) == int:
    print ("Variable x is integer")

if type(x) == float:
    print ("Variable x is float")

if type(x) == complex:
    print ("Variable x is complex")

if type(x) == str:
    print ("Variable x is string")
Variable x is complex


if else

To explore the use of if else we will take a number input, check whether it is positive or negative and then give its absolute value

n = input("Integer? ")
n = int(n)
if n < 0:
    print ("The absolute value of",n,"is",-n)
else:
    print ("The absolute value of",n,"is",n)

The absolute value of -9 is 9


Conditions with Logical Operators

sometimes we want to check two things together. we need logical operator and, or, not to combine two or more conditions

name = "John"
age = 23
if name == "John" and age == 20:
    print ("Good")
if name == "John" or age == 20:
    print ("Ok")
Ok


Another example using not operator.

std_list = ["Mark","Bob", "Alice","John"]
name = "Alex"
if name not in std_list:
    print ("You are not registered.")
You are not registered.


if elif else

If we have multiple conditions in our code then it is better to write it with if elseif structure.



name = "John"
age = 23
if name == "John" and age == 20:
    print ("Good")
elif name == "John" or age == 20:
    print ("Ok")
else:
    print ("Incorrect Info")
Ok


The output is same in both programs but there are performance differences. In our first code both conditions will run everytime. In our second code if any one condition gets true, rest will be skipped.


A Simple Game using while and if condition

If we have multiple conditions in our code then it is better to write it with if elseif structure.



answer = 23
question = 'What number am I thinking of? '
print ('Let\'s play the guessing game!')
while True:
    guess = int(input(question))
    if guess < answer:
        print ('Little higher')
    elif guess > answer:
        print ('Little lower')
    else: # guess == answer
        print ('YES!!!')
        break
