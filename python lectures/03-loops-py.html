Now we are familiar with list data type. suppose there is a list of 10 numbers and we want to take square of all numbers. if you dont know about loops you will write something like:

numbers = [2,10,3,6,25,8,12,1,9,50]

but what if the list contains 100 numbers. will you write hundred times print statement???...
The solution is a loop. When there is a need to repeat some piece of code n times, we use loop. There are two types of loop in python, For and While. We use For loop when we know in advance how many times the loop should run. The For loop contains an iterable method which process every item one by one. Here is the code to write the above script using For loop.



numbers = [2,10,3,6,25,8,12,1,9,50]
for i in numbers:
    print(i**2)

Double asteriks (**) are used for exponentiation. Here loop will iterate each number one by one and displays its square.


"in" keyword takes each item from list one by one and assigns it in i variable. Then we can use this variable in any way we want. The body of loop is usually indented with minimum four spaces. To make the concept more clear we have another example.

N= 10
sum = 0
print("Sum of Numbers from 0 to 9")
for num in range(N):
    sum = sum + num
print("Sum = ", sum)
Sum of Numbers from 0 to 9
Sum = 45


range function is used to generate a sequence of numbers, by default starts from zero and increment with one. So it will generate numbers from 0 to 9. The loop will iterate each number one by one and add this number into sum variable. Note that the last print statment will be run once only because it is outside the body of for loop.

Early exit

It is possible to exit a loop before completion using break statement.



for i in range(10):
    x = input("Enter a number less than 100")
    if int(x) >= 100:
        break
    print("x = ", x)

We may add an else clause at the end of for or while loop. When the loop completes it runs the else clause. This is actually a much better model for whatâ€™s going on in our loop. Reading Command Line arguements through for loop

Python has provided us some built in functionality to use like math functions sin, cos, sqrt etc. There is one built in class called sys through which we can use command line arguements. The built in varible which is used to store all command line arguement is sys.argv. This is an array. To access first element we write sys.argv[0], second element sys.argv[1] and so on. The first element contains the name of the script.



import sys
start = 0; stop = len(sys.argv); step = 1
for index in range(start, stop, step):
    print ("sys.argv[%d]=%s" % (index, sys.argv[index]))

Accessing two or more lists simultanously

Sometimes we need to access two or more lists simultanously. For this purpose we use zip function. which can iterate two or more lists simultanously.



xlist = [1, 2, 3, 4]
ylist = [2, 3, 2, 3]
zlist = [100, 200, 300, 100]
for x, y, z in zip(xlist, ylist, zlist):
    print(x**y+z)
