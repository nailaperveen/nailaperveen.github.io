Objectives

2. How to deal with Arrays and Vector Data

  • Duration: 20 min
  • Exercise : 1

NumPy is the fundamental package for scientific computing with Python. It contains among other things:

  • a powerful N-dimensional array object
  • sophisticated (broadcasting) functions
  • tools for integrating C/C++ and Fortran code
  • useful linear algebra, Fourier transform, and random number capabilities.

Arrays

An array in python is similar to a list but the main difference is that elements in a list can be of different data types while all the elements in arrays should be of same data type. This makes arrays more efficient because we know exactly that each item in an array will take the same storage space in memory. vectors and matrices are also called as arrays in NumPy. Vectors are usually represented as 1-dimensional array.


Arrays vs Lists

One of the main advantages of NumPy is its advantage in time compared to standard Python. Let's look at the following functions:
import time
size_of_vec = 1000

def pure_python_version():
   t1 = time.time()
   X = range(size_of_vec)
   Y = range(size_of_vec)
   Z = []
   for i in range(len(X)):
      Z.append(X[i] + Y[i])
   return time.time() - t1

def numpy_version():
   t1 = time.time()
   X = np.arange(size_of_vec)
   Y = np.arange(size_of_vec)
   Z = X + Y
   return time.time() - t1

t1 = pure_python_version()
t2 = numpy_version()
print(t1, t2)
print("Numpy is in this example " + str(t1/t2) + " faster!")
0.00030875205993652344 2.0503997802734375e-05
Numpy is in this example 15.05813953488372 faster!

Creation of a vector with Evenly Spaced Values using “ArrayRANGE” and "linspace":

The syntax of arange: arange([start,] stop[, step,], dtype=None)
arange returns evenly spaced values within a given interval. The values are generated within the half-open interval '[start, stop)'. arange returns an ndarray. If the 'start' parameter is not given, it will be set to 0. The end of the interval is determined by the parameter 'stop'. Usually, the interval will not include this value. The spacing between two adjacent values of the output array is set with the optional parameter 'step'. The default value for 'step' is 1. The type of the output array can be specified with the parameter 'dtype'.


The syntax of linspace: linspace(start, stop, num=50, endpoint=True, retstep=False) linspace returns an ndarray, consisting of 'num' equally spaced samples in the closed interval [start, stop]

import numpy as np
x = np.arange(0, 5, 0.5)
print (x)
y = np.linspace(0, 5, 10)
print (y)

Operations on Arrays

Note that once we have created an array we can perform calculations on every element of array with a single statement
import numpy as np
a = np.array([20,30,40,50])
b = np.arange(4)
print("a = ", a)
print ("b = ", b)
c = a-b
print ("a-b = ", c)
print("b**2", b**2)
print("10*np.sin(a) = ", 10*np.sin(a))
print("a<35 = ", a<35)

Matrix Multiplication

Using numpy.random we can create arrays containing random values. Two arrays can be multiplied in the usual linear-algebra way using numpy.matrixmultiply.

Eigenvectors and Eigenvalues

Here is a small example that computes the [trivial] Eigenvectors and Eigenvalues (eig) of the unity matrix (eye)):

A = np.eye(3)

evalues , evectors = LA.eig(A)

print ("evectors = ", evectors)

Indexing and Slicing

Assigning to and accessing the elements of an array is similar to other sequential data types of Python, i.e. lists and tuples. Indexing and Slicing in one dimensional arrays



import numpy as np
A = np.array([11,12,13,14,15])
print ("Index : ", A[1])
print("Slice : A[1:3] = ", A[1:3])
print("Slice : A[:3] = ", A[:3])
print("Slice : A[1:] = ", A[1:])
print("Slice : A[:] = ", A[:])
Index : 12
Slice : A[1:3] = 12, 13
Slice : A[:3] = 11,12,13
Slice : A[1:] = 12,13,14,15
Slice : A[:] = 11,12,13,14,15
