An Interpreted Language

Python can run on any platform. On a Unix machine usually the Python interpreter is installed in /usr/local/bin folder. On a Windows machine the defualt place for Python interpreter is C:\. But the location is a part of installation process so you can change its place but it is not recommended. If you change the location of Python interpreter then there is a need to update the PATH variable. To start Python you just type python in terminal window.



Interactive Mode

Three greater-than signs (>>>) is the primary prompt of Python. when we run Python using python command, it is started in interactive mode. First it shows the version information and then display the primary prompt. Primary prompt(>>>) means interpreter is ready to accept commands and run them immediately. when a statement breaks into two lines it shows the secondary prompt which is tripple dot (...)

Where to keep the data?

We do programming to organize our data; so first we will see what is data? and how to organize it?
As we have discussed before data can be anything, a number, a name, some text or image. To perform processing on our data(numbers and strings), we need them to store somewhere. That is where the concepts of variables come in. Variables are simple containers to hold some data in memory. A variable can hold any thing: Number, String, File, image, video etc. In Python there is no need to define the type of variable. We just declare the name of a variable and pass some value to it. Depending on the value passed to the variable, the interpreter determines the type of variable automatically and it can be changed at any time. For example there is a variable "price". Using interactive mode, we can assign value in it like this. The data type of variable price is Number here.



>>> price = 80

and later in our code we may use the same variable like this. Now the data type of variable price is String.

>>> price = "eighty"


Here in this example first we assigned a numeric value and then we assigned a string value in the same variable. It is acceptable in Python. There will be no error. But using variables like this is a bad practice, as we may confused ourselves later in our code or it may bring inconsistency at run time.

Remember: To start python interactive mode type python in terminal from anywhere. It will start python in interactive mode.



In python interactive mode when we define a variable, solve any equation or import any library, it keeps a record of every statement. for example if we define a variable x on first line, then in the second line we can use this variable

>>> x = 0.5

Display Data

print statement in Python is used to diplay output on screen. It can take zero or more parameters. With zero parameters it displays a blank line. If there are more than one parameter, they are separated with commas. We can also perform arithemtic in print statment.

>>> print("5000 Kbs is equal to ", 5000/1000 , " Mbs")
5000 Kbs is equal to 5 Mbs


To display a value from variable give the name of variable without quotation marks.

>>> a = 50
>>> print(a)
50


We can use escape sequences inside print function. An escape sequence is a sequence of characters that does not represent itself when used inside a character or string literal, but is translated into another character or a sequence of characters that may be difficult or impossible to represent directly.

>>> a = 50
>>> b = 10
>>> print("a = ",a , "\nb =", b, "\nsum = ",a+b)
a = 50
b = 10
sum = 60

Formatted Print

The format specifier of type {:W.Df} means that a Float should be printed with a total Width of W characters and D digits behind the Decimal point.

>>> print ("a = {:.2f}".format(32.123456))
a = 32.12


To define a total width of 6 characters with 2 digits after decimal points.

>>> print ("a = {:6.2f}".format(32.123456))
a =    32.12
